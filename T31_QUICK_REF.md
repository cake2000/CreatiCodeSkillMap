## Key Learning Progressions

### Multiplayer Journey (Grade 5-7)
```
Grade 5: Create → Join → List → Status
Grade 6: Add Sprites → Remove → List Players → Reset
Grade 7: Movement → Broadcast → Collisions → Server Model
```

### Data Storage Journey (Grade 6-7)
```
Grade 6: Google Sheets (Read/Write/Manage)
Grade 7: Database (Insert/Query/Update/Remove)
```

### Cloud Concepts Journey (Grade 5-8)
```
Grade 5: Basic web fetching
Grade 6: HTTP/HTTPS, Google Sheets
Grade 7: Database, Network architecture
Grade 8: AI cloud integration, Security
```
