## One Skill = One Block Principle

**Before Optimization:**
- T25.G3.00.01: "Create and name variables" covered create, set, AND display (3 blocks)
- T25.G3.06.01: "Create a simple table" covered create, add row, AND display (3 blocks)

**After Optimization:**
- Each skill focuses on ONE specific block or concept
- Example: T25.G3.00.01.01 (create), T25.G3.00.01.02 (set), T25.G3.00.01.03 (display)
- Exception: Conceptual skills that don't map to single blocks (planning, design, ethics)

## Benefits of Optimization

1. **Clearer Learning Path**: Students master one concept at a time before combining
2. **Better Assessment**: Teachers can verify specific skill mastery (e.g., "Can student create a list?" vs vague "Can student work with lists?")
3. **Easier Debugging**: When students struggle, teachers can pinpoint exact missing skill
4. **Complete Block Coverage**: All CreatiCode data blocks now have corresponding skills
5. **AI Integration Ready**: Clear pathway from data structures to AI/ML applications
6. **Real-World Preparation**: Skills match professional data engineering workflows

## Implementation Notes for Teachers

### Pacing Recommendations
- **Grade 3**: 2-3 skills per week (foundational, go slowly)
- **Grade 4**: 3-4 skills per week (building on G3 foundation)
- **Grade 5**: 4-5 skills per week (more independent practice)
- **Grade 6-8**: 3-4 skills per week (more complex, need processing time)

### Common Pitfalls to Avoid
1. **Don't skip sub-skills**: Even if they seem simple, each builds essential muscle memory
2. **Verify monitors**: Many G3 students forget to display monitors (G3.00.01.03, G3.00.02.03)
3. **Test with edge cases**: Empty lists/tables cause errors (teach defensive programming)
4. **Emphasize naming**: Good variable/list/table names prevent confusion later

### Project Integration
- **Capstone opportunities**:
  - G5: Build a game with full state management (G5.01 series)
  - G6: Create a data dashboard with CSV export (G6.07-08 series)
  - G7: Build a collaborative leaderboard (G7.05 series)
  - G8: Train an AI classifier from collected data (G8.05 series)
