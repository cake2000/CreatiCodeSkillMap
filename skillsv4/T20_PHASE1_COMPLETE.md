### 2. New Skills Added (8 total)

All new skills fill critical gaps in CreatiCode feature coverage:

#### **T20.G5.01.01** - Map data to two visual properties
- **Purpose:** Scaffolding bridge between 1D (G5.01) and multi-field (G6.04)
- **What it teaches:** Map data to TWO visual properties (e.g., x-position and color)
- **Dependencies:** T20.G5.01, T10.G4.02
- **Impact:** Fills jump from 1-property to 3-property visualization

#### **T20.G6.05.01** - Apply materials and textures to 3D art
- **Purpose:** Teach PBR materials (metallic, roughness, textures)
- **What it teaches:** configure material properties, apply textures, surface appearance
- **Dependencies:** T20.G5.04.01
- **Impact:** Covers materials category (15+ blocks)

#### **T20.G6.05.02** - Create 3D curve and line art
- **Purpose:** Teach 3D lines/curves for wireframe art
- **What it teaches:** Create point lists, draw 3D curves, wireframe structures
- **Dependencies:** T20.G5.04.01, T20.G4.01
- **Impact:** Covers 3D line/curve blocks (8+ blocks)

#### **T20.G6.05.03** - Create interactive 3D generative art
- **Purpose:** Extend 2D interactivity (G5.03) to 3D
- **What it teaches:** Mouse/key events control 3D parameters, camera, or generation
- **Dependencies:** T20.G5.03, T20.G5.04.01
- **Impact:** Interactive 3D art pathway

#### **T20.G7.04.01** - Create particle-based generative art
- **Purpose:** Teach particle systems as artistic tool
- **What it teaches:** Configure emitters, color gradients, size-over-lifetime, movement
- **Dependencies:** T20.G6.05.01, T20.G5.02
- **Impact:** Covers particle category (18+ blocks)

#### **T20.G7.05.02** - Use lighting to enhance 3D algorithmic art
- **Purpose:** Teach lighting as artistic technique
- **What it teaches:** Add/configure lights (point, directional, spot), position, color, intensity
- **Dependencies:** T20.G5.04.01, T20.G6.05.01
- **Impact:** Covers lighting category (7+ blocks)

#### **T20.G7.05.03** - Generate custom 3D shapes from calculated vertices
- **Purpose:** Teach procedural geometry creation
- **What it teaches:** Calculate vertex positions, create shapes from lists, extrusion basics
- **Dependencies:** T20.G6.05.02, T10.G6.01
- **Impact:** Advanced 3D creation technique

#### **T20.G8.05.01** - Apply post-processing effects to generative art
- **Purpose:** Teach post-processing pipeline
- **What it teaches:** Configure bloom, glow, blur effects, artistic enhancement
- **Dependencies:** T20.G7.05.02, T20.G8.04
- **Impact:** Covers post-processing category (5+ blocks)
