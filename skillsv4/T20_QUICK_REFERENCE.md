## Common Learning Sequences

### Sequence A: Traditional 2D Art Path
```
K → G1 → G2 → G3 (pen/shapes) → G4 (patterns) →
G5 (animation) → G6 (math) → G7 (generative) → G8 (expert)
```

### Sequence B: 3D-Focused Path
```
K → G1 → G2 → G3 (drawing basics) → G4 (loops/custom blocks) →
G5 (3D setup + shapes) → G6 (3D materials) →
G7 (3D lighting + particles) → G8 (3D advanced)
```

### Sequence C: Data Visualization Path
```
K → G1 → G2 → G3 (drawing basics) → G4 (data lists) →
G5 (bar charts) → G6 (multi-field) → G8 (multi-dimensional)
```

### Sequence D: Generative Art Path
```
K → G1 → G2 → G3 (basics) → G4 (patterns) →
G5 (fractals) → G6 (math functions) →
G7 (L-systems, cellular automata) → G8 (constrained, hybrid)
```

## Troubleshooting Common Skill Gaps

**Problem:** Student can't do G4.01 (spirals)
- Check: Do they understand G3.13 (variables)?
- Check: Can they use T09.G3.02 (change block)?

**Problem:** Student can't do G5.11 (3D patterns)
- Check: Did they complete G5.07-10 in order?
- Check: Can they use conditionals (T08.G3.01)?

**Problem:** Student can't do G6.05 (multi-field viz)
- Check: Did they master G5.01-02 (basic viz)?
- Check: Do they understand nested lists (T10.G5.01)?

**Problem:** Student can't do G7.10 (L-system trees)
- Check: Did they complete G7.09 (string generation)?
- Check: Can they use trig functions (G6.06-07)?

**Problem:** Student can't do G8.10 (capstone)
- Check: Did they complete at least 3 of: G8.06-09?
- Check: Did they complete G8.02 (constrained art)?
