KEY BLOCK PATTERNS
==================

Creating Widgets (position-based):
- Most widgets use: at X (X) Y (Y) ... as [NAME]
- Position is centered at given coordinates
- NAME parameter is required and cannot be empty

Getting Widget Values:
- Use: value of widget [WIDGETNAME v]
- Works for textboxes, buttons, date/color pickers, sliders, dropdowns, etc.
- Different widgets return different value types

Setting Widget Values:
- Use: set value to [V] for widget [WIDGETNAME v]
- Works for most widget types
- Format depends on widget type (text, date YYYYMMDD, color #RRGGBBAA, etc.)

Animation Blocks:
- All animation blocks include duration and blocking mode
- Blocking = wait until complete; Non-blocking = continue immediately

Event Blocks (Hat blocks - no inputs):
- widget_whenwidgetclicked
- widget_whenwidgetchanges
- widget_whenmouseenter / widget_whenmouseleave
- widget_whentabselected
- All 4 video event blocks

Color Format: #RRGGBBAA
- Red, Green, Blue, Alpha (transparency)
- Examples: #FF0000FF (red), #00FF00FF (green), #0000FFFF (blue)

Date Format: YYYYMMDD
- No separators
- Example: 20210620 for June 20, 2021
