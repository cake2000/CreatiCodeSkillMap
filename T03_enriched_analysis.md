# T03 – Problem Decomposition & Project Planning: Dependency Analysis

## Age-Appropriateness Framework for This Topic

**K-2 (Ages 5-8)**: Concrete, hands-on decomposition of familiar objects and routines. Very short tasks (5-10 min). Visual, tangible examples.

**3-5 (Ages 8-11)**: Beginning to plan multi-step projects. Can handle 15-20 min planning tasks. Transitioning from physical to digital planning.

**6-8 (Ages 11-14)**: Abstract planning skills, complex multi-feature projects, professional documentation practices. Can sustain 30+ min analysis.
