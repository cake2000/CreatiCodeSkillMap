TEXT-TO-SPEECH BLOCKS
================================================================================

Block ID: ai_speakinlanguage
Syntax: say [TEXT] in [LANGUAGE v] as [VOICETYPE v] speed (SPEEDRATIO) pitch (PITCHRATIO) volume (VOLUMERATIO) store sound as [SOUNDNAME]
API: Microsoft Azure Text-to-Speech

Block ID: ai_stopspeaking
Syntax: stop speaking
API: N/A


SPEECH RECOGNITION BLOCKS (MICROSOFT AZURE)
================================================================================

Block ID: ai_startspeech
Syntax: start recognizing speech in [LANGUAGE v] record as [SOUNDNAME]

API: Microsoft Azure Speech Recognition

Block ID: ai_stoprecognizer

IMAGE GENERATION BLOCKS (DALL-E)
================================================================================

Syntax (Reporter): OpenAI DALL-E: generate image with request [DESCRIPTION] resolution [RESOLUTION v]

================================================================================
FEATURES & CUSTOMIZATION
================================================================================

TEXT-TO-SPEECH CUSTOMIZATION:
- Speed: Percentage value (e.g., 100% = normal speed)
- Pitch: Percentage value (e.g., 100% = normal pitch)
- Volume: Percentage value (e.g., 100% = normal volume)
- Optional: Store as sound clip for later playback

SPEECH RECOGNITION VARIANTS:
- Single recognition: Record once and get full text
- Continuous recognition: Stream real-time recognition results into a list
- Optional: Store recorded audio as sound clip

IMAGE GENERATION:
- Reporter mode: Returns URL of generated image
- Command mode: Generates and saves as costume directly
- Three resolution options for different use cases

CONTENT MODERATION:
- Text moderation: Check for inappropriate content
- Image URL moderation: Check images from URLs
- Local image moderation: Check costumes from project
