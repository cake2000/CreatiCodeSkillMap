## K-2 FOUNDATIONS (12 new skills)

### Kindergarten (3 skills)

**T24.GK.07** - Data tells a story about real things
- Connects data to real-world scenarios
- Matching activity: data cards → picture scenarios
- Learning: Data represents real information

**T24.GK.08** - Choose the right representation for a purpose
- Picture vs word vs numeral based on question
- MCQ: 5 purpose-based scenarios
- Learning: Different representations serve different purposes

**T24.GK.09** - Spot when data doesn't match reality
- Identify picture-label mismatches
- Tap incorrect pairs (6 total, 3 wrong)
- Learning: Data should accurately represent reality

### Grade 1 (3 skills)

**T24.G1.07** - Explain why we organize data into groups
- Compare sorted bins vs scattered pile
- Audio/text explanation with visual comparison
- Learning: Organization makes data more useful

**T24.G1.08** - Collect real data from the classroom
- Conduct actual classroom survey with tally marks
- Blended physical/digital activity
- Learning: Data comes from observing real events

**T24.G1.09** - Predict data before collecting it
- Two-stage: prediction → collection → comparison
- Prediction drag interface + actual data recording
- Learning: We can make predictions about data

### Grade 2 (6 skills)

**T24.G2.08** - Identify what question the data answers
- REVERSE: Given data → identify question it answers
- Match data displays to questions (3 scenarios)
- Learning: Data is collected to answer specific questions

**T24.G2.09** - Explain why this data representation was chosen
- Match representations to purposes with justification
- 3 parallel examples (list/table/timeline)
- Learning: Representation choices have reasons

**T24.G2.10** - Spot incomplete data and suggest what's missing
- Identify gaps in 4 incomplete displays
- Select appropriate fixes from options
- Learning: Complete data includes all necessary information

**T24.G2.11** - Compare two ways to show the same data
- Side-by-side display comparison
- Checklist of advantages for each
- Learning: Different representations have different strengths

**T24.G2.12** - Verify that data representation is accurate
- Trace raw data → chart representation
- Mark each value as verified or error
- Learning: We can verify data was represented correctly

**T24.G2.13** - Create data to match a representation
- REVERSE: Given chart → create matching data
- Draw/drag correct quantities to match chart
- Learning: Bidirectional data-representation relationship

## COMPRESSION/ENCODING DEPTH (6 new skills)

**T24.G4.18** - Encode text using simple substitution cipher
- Full alphabet substitution (A→M, B→N, etc.)
- Encode/decode 3-letter words using cipher table
- Learning: Systematic encoding schemes can represent entire alphabets

**T24.G5.15** - Compare fixed-length vs variable-length encoding
- Calculate storage for sample text using both approaches
- Fixed (every letter = 8 bits) vs variable (common = fewer bits)
- Learning: Variable-length encoding can save space

**T24.G6.15** - Implement Huffman-style frequency-based encoding
- Analyze letter frequency → assign code lengths
- Frequency table → code design → compression calculation
- Learning: Frequency analysis enables efficient compression

**T24.G6.16** - Explain lossy vs lossless compression with examples
- Compare lossy (JPEG) vs lossless (ZIP)
- Categorize data types by appropriate compression type
- Learning: Some compression loses information; choice depends on data importance

**T24.G7.13** - Implement dictionary-based compression (LZW concept)
- Repeated patterns → shorter codes
- Build dictionary + encode text with pattern replacement
- Learning: Dictionary compression exploits repeated patterns

**T24.G8.17** - Analyze compression tradeoffs: speed vs ratio vs quality
- Compare algorithms on 3 dimensions
- Select best algorithm for different scenarios with reasoning
- Learning: Compression algorithms have multi-dimensional tradeoffs

## SKILL ID PATTERNS

### New Skill Naming Convention

**Pattern 1: Sequential additions within grade**
- T24.GK.07, T24.GK.08, T24.GK.09 (after existing GK.01-06)
- T24.G1.07, T24.G1.08, T24.G1.09 (after existing G1.01-06)
- T24.G2.08-13 (after existing G2.01-07)

**Pattern 2: Bridge skills**
- T24.G3.00.BRIDGE.01-05 (explicit bridge sequence)
- Fits between G3.00 and G3.00.01.01

**Pattern 3: Thinking skills**
- T24.G3.10, G4.13, G4.14, G5.11, G5.12, G6.11, G7.10, G8.12
- Numbered after existing skills in each grade

**Pattern 4: Debugging skills**
- T24.G3.11, G3.12, G4.15, G4.16, G5.13, G6.12, G7.11, G8.13, G8.14
- Numbered after existing skills in each grade

**Pattern 5: Consolidated skills**
- T24.Gx.xx.NEW (explicitly marked as consolidated)
- Replaces multiple original skill IDs
