## Implementation Notes

### When to Apply These Fixes:

1. **HIGH PRIORITY:** Skills that explicitly mention the concept
   - Example: "conditional" in title → needs T09.G3.03
   - Example: "nested loops" in title → needs T10.G3.05

2. **MEDIUM PRIORITY:** Skills that likely use the concept
   - Example: "patterns" might need loops
   - Example: "parameters" needs procedure skills

3. **LOW PRIORITY:** Skills that might indirectly benefit
   - Review on case-by-case basis

### When to Skip:

- Dependency is already present (check current dependencies)
- Concept is not actually used in the skill
- Within-topic dependency already covers it
- Skill is purely conceptual/theoretical
