## Application Topics: Programming Foundation Analysis

All application topics (T14-T24) show consistent patterns in their programming dependencies.

**T06 (Events)**: 100% of all Grade 3+ application skills

**T09 (Variables)**: 100% of all Grade 3+ application skills

**T07 (Loops)**: 67-92% across application topics

- Animation, iteration, and repetition are core to most applications

**T08 (Conditionals)**: 71-100% across application topics

### Advanced Dependencies (50% prevalence in Grades 6-8)

- Consistent 50% pattern shows systematic enrichment

**T12 (Program Organization)**: 50-55% across application topics

**T13 (Testing/Debugging)**: 50-59% across application topics

**Characteristics**:
- Average: 0.60-1.44 dependencies
- Mostly within-topic progressions
- Minimal cross-topic dependencies
- Focus: Build foundational concepts without overwhelming prerequisites


- Standardized core programming foundation
- Focus: Combine multiple concepts for first time
