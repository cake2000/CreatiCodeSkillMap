## Common Student Errors - Fast Reference

| Error | Quick Fix |
|-------|-----------|
| **Blocks not connected** | Listen for "click" sound when snapping |
| **Green flag doesn't work** | Event block must be at TOP |
| **Variable doesn't change** | SET before CHANGE |
| **Collision not detected** | Make sprites BIGGER, check spelling |
| **Forever loop stuck** | Click red STOP button |
| **X and Y mixed up** | X = left/right, Y = up/down |
| **Code deleted** | Check undo button, check saved projects |

## Daily Lesson Flow Template

**Opening (5 min):**
- Review previous skill
- Introduce today's skill
- Show real-world example

**Demo (10 min):**
- Teacher demonstrates
- Think-aloud process
- Show common errors

**Guided Practice (10 min):**
- Students follow along
- Check for understanding
- Fix errors together

**Independent Work (15 min):**
- Students practice
- Teacher circulates
- Peer helpers active

**Closure (5 min):**
- Share student work
- Exit ticket
- Preview tomorrow

**Total:** 45 minutes per skill

## Red Flags - When to Get Help

**For Individual Students:**
- Stuck on same skill for 3+ days
- Giving up easily
- Off-task consistently
- Can't explain completed code
- Copying without understanding

**For Whole Class:**
- <60% mastery on any skill
- <50% mastery on gateway skill
- High frustration levels
- Low engagement
- Behind by 1+ week

**Action:** Contact curriculum specialist, tech support, or admin
