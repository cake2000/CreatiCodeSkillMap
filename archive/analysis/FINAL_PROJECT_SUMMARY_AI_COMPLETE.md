## üéì KEY IMPLEMENTATION INSIGHTS

### **For K-2 (Ages 5-8)**

**Approach:** Picture-based, concrete, manipulative
**Duration:** 10-15 minutes per skill
**Modality:** Online but NO coding required
**Success Factors:**
- Audio support (pre-readers)
- Large, colorful visuals
- Immediate feedback
- Unlimited retries
- Familiar contexts (animals, food, home, school)

**Critical K-2 AI Concepts:**
- Pattern recognition (computers learn from patterns)
- Sensing (cameras see, microphones hear)
- Binary decisions (yes/no, this/that)
- Human creators (people make AI)

### **For Grade 3 (THE Gateway)**

**Approach:** Block coding begins, foundational skills
**Duration:** 40-50% of instructional time on 5 gateway skills
**Modality:** CreatiCode block-based programming
**Success Factors:**
- Ensure mastery before advancing
- Multiple practice opportunities
- Frequent formative assessment
- Small group intervention for struggling students
- Real-world connections

**5 Critical Gateway Skills:**
1. T06.G3.01 - Event handlers
2. T07.G3.01 - Loops
3. T08.G3.01 - Conditionals
4. T09.G3.01 - Variables ‚≠ê MOST IMPORTANT
5. T13.G3.01 - Debugging

### **For Grades 4-8 (Application & Mastery)**

**Approach:** Project-based, integrated, student-driven
**Duration:** 2-4 week projects integrating multiple skills
**Modality:** CreatiCode with 2D, 3D, AI, physics extensions
**Success Factors:**
- Student choice (pathways: games, data, AI, etc.)
- Peer collaboration
- Ethical considerations integrated
- Competition preparation (optional)
- Portfolio development

**Critical AI Progression:**
- **G3-4:** Using AI (media tools, chatbots)
- **G5-6:** Understanding AI (how it learns, representations)
- **G7-8:** Creating AI ethically (bias testing, model cards, fairness)
