# T05 – Human-Centered Design & Simulation Planning: Dependency Analysis

## Age-Appropriateness Framework for This Topic

**K-2 (Ages 5-8)**: Concrete empathy - who do tools help? Very simple improvements. Focuses on immediate family/classroom context.

**3-5 (Ages 8-11)**: User stories, simple personas, basic UX flows. Can think about "someone" abstractly. Beginning simulation concepts.

**6-8 (Ages 11-14)**: Formal HCD principles, accessibility audits, bias analysis, controlled experiments. Ready for abstract ethical reasoning and complex trade-offs.

## Summary

**Total Skills**: 35 (average of ~3.9 per grade; some grades have 3, some have 4)
**Granularity**: ✓ All appropriately sized
**Age-Appropriateness**: ✓ **EXCELLENT** progression

**Issues Found**: None - all dependencies properly ordered

**Age-Appropriateness Highlights** (Perfect for HCD topic):
- **K-2**: Very concrete empathy (family, classroom), simple tools, binary judgments
- **3-5**: User stories, personas, UX flows - can think about abstract "users"
- **6-8**: Formal HCD principles, accessibility audits, bias analysis, scientific simulations

**Interdisciplinary Nature**:
This topic uniquely bridges:
- **Design thinking** (empathy, iteration, usability)
- **Ethics** (accessibility, bias, harm mitigation)
- **Science** (simulation design, experimentation, model validity)

**Cross-Topic Dependencies**:
- **T01 (Algorithms)**: Algorithm design and analysis
- **T02 (Representing)**: Flowcharts for UX flows
- **T03 (Decomposition)**: Project planning and structure
- **T06 (Events)**: Code context for programs being evaluated
- **T13 (Testing)**: Testing and iteration methods
- **T27 (Data Analysis)**: Simulation data analysis
- **T28 (Chance/Sampling)**: Probability in simulations
- **T35 (Impacts)**: **CRITICAL** for ethics and societal impact

**Key Progression Themes**:
1. **Empathy development**: Who → specific users → personas → diverse audiences
2. **Design formality**: Ideas → plans → documents → formal HCD
3. **Iteration**: Suggest → improve → test → multi-round refinement
4. **Accessibility**: Easy/hard → different needs → audit → bias analysis
5. **Simulation**: Simplify → plan → design → experiment → validity

**Unique Strengths**:
- Most age-appropriate progression of any topic reviewed so far
- Natural integration of ethics throughout (not bolted on)
- Simulation skills build systematically from G5-G8
- Perfect scaffolding from concrete to abstract

**Recommendation**: **NO CHANGES NEEDED** - this topic is exceptionally well-designed for K-8 progression.
