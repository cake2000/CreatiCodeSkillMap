## CreatiCode-Specific Syntax

- Block creation: `define (BlockName (param))`
- Parameter access: `argument (param)`
- Command call: `call BlockName [value]`
- Reporter call: `report BlockName [value]`
- Return value: `return [value]`
- Comment: `// [comment]`
