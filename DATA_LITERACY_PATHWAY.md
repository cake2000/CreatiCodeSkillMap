# Complete Data Literacy Pathway: K-8 Data Science Curriculum

**Based on:** Topics T25-T29 Comprehensive Dependency Analysis
**Date:** 2025-11-17
**Target Audience:** K-8 educators implementing data science curriculum

## Pathway Overview

The K-8 Data Literacy Pathway maps a complete progression from simple data awareness to professional-grade data science practices. This pathway is built on 167 carefully sequenced skills across 5 integrated topics.

### The Big Picture: Data Science Pipeline
```
REPRESENT → COLLECT → ANALYZE → EXPERIMENT → PROCESS TEXT
   (T25)    (T26)     (T27)      (T28)       (T29)
```

This mirrors professional data science workflow and supports learning transfer to real applications.

- T26.G1.01: Surveys and questions
- T27.G1.01-02: Counting and pictographs
- T28.G1.01: Likelihood concepts
- T29.G1.01-02: Counting words

### Grade 2 Skills
- T25.G2: Variables and tables
- T26.G2: Simple data collection
- T27.G2: Bar charts and comparisons
- T28.G2: Fair games and experiments
- T29.G2: Word operations

## Stage 2: Data Structures & Computational Foundations (Grade 3-4)

### Duration: ~2 years of integrated instruction

### Critical Gateways
1. **T25.G3.05: Build a data structure with a list (Grade 3)**
   - Unlocks all structured data work
   - Requires 2-3 weeks focused instruction
   
2. **T29.G3.03: Ask AI a question and receive response (Grade 3)**
   - Unlocks AI-powered NLP skills
   - Requires 1-2 weeks focused instruction

### Grade 3 Skills
- T25.G3: Data types and lists (GATEWAY)
- T26.G3: Multi-column tables
- T27.G3: Simple statistics (average)
- T28.G3: Simulations with code (50+ trials)
- T29.G3: String operations and AI (GATEWAY)

### Grade 4 Skills
- T25.G4: Tables and binary representation
- T26.G4: Event logging in games
- T27.G4: Mode, median, comparisons
- T28.G4: Theoretical probability
- T29.G4: String length, AI sentiment

### Learning Outcomes (End of Grade 4)
Students can:
- Create and manipulate lists programmatically
- Store and retrieve data by index
- Calculate basic statistics (average, mode, median)
- Run simulations (50+ trials) using loops
- Create labeled charts and graphs
- Use AI to answer questions about text

## Stage 4: Professional Data Science Projects (Grade 7-8)

### Duration: ~2 years of capstone-focused instruction

### Grade 7 Capstone Skills (3 major capstones)

**T26.G7.03: Manipulate data with code (CAPSTONE)**
- Filter by multiple conditions
- Sort, aggregate, calculate derived fields
- 3-4 weeks, complex real dataset

**T28.G7.02: Monte Carlo simulation (CAPSTONE)**
- Estimate Pi or solve probability problem
- Run 10,000+ simulations
- 3-4 weeks, teaches powerful technique

**Plus Advanced Skills:**
- T25.G7: Advanced structures, floating point, missing data
- T27.G7: Simulations, bias, trends, distributions
- T29.G7: Text preprocessing, similarity, AI Q&A

### Grade 8 Capstone Skills (6 major capstones)

**T26.G8.02: Complete data investigation (MAJOR CAPSTONE)**
- Question → Collect → Clean → Analyze → Conclude
- 4-5 weeks, 7 dependencies (highest complexity)

**T27.G8.01: Data analysis investigation (CAPSTONE)**
- Choose question, find/collect data, analyze, visualize, conclude
- 4-5 weeks

**T27.G8.03: Interactive data dashboard (MAJOR CAPSTONE)**
- Web-based dashboard with filters and real-time visualizations
- 4-5 weeks, bridges to UI design (T16)

**T28.G8.01: Real-world experiment design (CAPSTONE)**
- Hypothesis, protocol, controls, data collection, analysis
- 4-5 weeks, partner with science

**T29.G8.01: Keyword search system (CAPSTONE)**
- Build indexed search engine
- 3-4 weeks, classic information retrieval

**T29.G8.04: Ethical implications of NLP (CAPSTONE)**
- Bias, privacy, misuse, accessibility in NLP
- 2-3 weeks, ethical focus

### Grade 8 Integration Skills
- T25.G8: Data schemas, bias, advanced structures
- T26.G8: Ethical collection, bias recognition
- T27.G8: Causal analysis, conclusions evaluation
- T28.G8: Probability estimation, AI bias analysis, ethics
- T29.G8: Sentiment analysis, entity extraction, NLP ethics

### Learning Outcomes (End of Grade 8)
Students can:
- Design and conduct complete data science investigations
- Build interactive data applications
- Implement sophisticated algorithms (Monte Carlo, search)
- Analyze variable relationships and causation
- Recognize and address bias in data and AI
- Make ethical decisions about data collection and use
- Communicate findings effectively
- Apply data science to real-world problems

## Essential Sequence

### Critical Path for Maximum Impact

**K-2: Build data awareness (low cognitive load)**
→ **G3-4: Introduce programming for data (mastery of gateways)**
→ **G5-6: Real research projects (computational thinking)**
→ **G7-8: Professional capstones (agency and ownership)**

### Gateway Mastery Timeline
- **Grade 3, Week 10-12:** T25.G3.05 (Lists) - intensive instruction
- **Grade 3, Week 28-30:** T29.G3.03 (AI Q&A) - introduction
- **Ongoing:** Reinforcement in grades 4-8

### Capstone Timeline
- **Grade 6, Weeks 20-24:** T26.G6.05 (Data Cleaning) - first capstone
- **Grade 7, Weeks 10-15:** T26.G7.03 (Data Manipulation) - second capstone
- **Grade 7, Weeks 16-20:** T28.G7.02 (Monte Carlo) - probability capstone
- **Grade 8, Quarters 1-2:** Multiple capstones (T26.G8.02, T27.G8.01, T27.G8.03)
- **Grade 8, Quarters 3-4:** Remaining capstones (T28.G8.01, T29.G8.01, T29.G8.04)

## Teacher Support by Stage

### K-2 Teacher Needs
- Real objects and manipulatives
- Engaging topics (pets, food, games)
- Simple data visualization tools
- Enthusiasm for exploration

### G3-4 Teacher Needs
- Block-based coding expertise (Scratch)
- Understanding of data structures
- Patience with logical thinking development
- Support for T25.G3.05 gateway mastery (2-3 weeks)
- Introduction to AI tools for T29.G3.03

### G5-6 Teacher Needs
- Spreadsheet proficiency
- Python or JavaScript basics
- Access to real, age-appropriate datasets
- Understanding of statistical concepts
- Project management skills
- Data visualization tools

### G7-8 Teacher Needs
- Advanced programming ability
- Statistical thinking
- Web development (for dashboards)
- Research methodology expertise
- Ethics discussion facilitation
- Mentoring complex projects
- Collaboration with science/social studies teachers

## Conclusion

The Data Literacy Pathway provides a complete K-8 progression that:
- **Progresses logically:** Concrete → Abstract, Simple → Complex
- **Reflects practice:** Mirrors real data science workflows
- **Integrates fully:** Builds on programming core, connects to AI
- **Emphasizes ethics:** Responsibility throughout
- **Provides projects:** 9 capstones for authentic learning

Students who complete this pathway will be prepared for high school computer science, equipped with data literacy for any field, and ready to make ethical decisions in our data-driven world.
