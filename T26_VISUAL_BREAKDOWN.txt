CREATICODE BLOCKS USED:
========================
Survey/Input:
  - ask and wait, answer, stop asking

Variables:
  - set/change/reduce variable
  - export variable, import variable
  - variable monitors

Lists:
  - add/insert/delete item
  - sort/reverse/reshuffle
  - join list, split text into list

Tables (Primary for G4+):
  - create table, add row, set cell
  - delete rows/columns
  - sort/aggregate/pivot
  - export/import CSV
  - find row by value

Database (Cloud - G5+):
  - insert from table into collection
  - fetch from collection with filters
  - update/delete documents
  - Boolean operators (and/or/not)
  - Query conditions

Game Features:
  - record player score
  - show/hide leaderboard
  - store/read user data

Debugging:
  - print to console (with color)
  - get console log

Sensors:
  - microphone loudness
  - mouse x/y position
  - timer (reset, pause, resume)

AI Vision:
  - face detection → table
  - hand detection → table
  - body pose detection → table

Advanced:
  - Semantic database (embeddings)
  - XO AI assistant

TO VERIFY:
  - Table export/import blocks
  - Google Sheets integration
