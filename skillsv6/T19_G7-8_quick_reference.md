## KEY TECHNIQUES BY GRADE

### Grade 7 Techniques:
- **L-Systems**: String rewriting, turtle graphics, fractal trees
- **Cellular Automata**: Rule lookup, evolution, 2D patterns (Game of Life)
- **Agent Systems**: Autonomous agents, flocking (Reynolds algorithm)
- **Particles**: Emitters, color gradients, size changes, algorithmic positioning
- **3D Lighting**: Point/directional/ambient lights, algorithmic positioning, mood
- **Interactive**: Sound-reactive, body tracking, gestures, AI sprites, physics
- **Design**: Color theory, composition, parameter studies

### Grade 8 Techniques:
- **Noise**: Value noise, multi-octave (fbm), flow fields
- **Data Viz**: Multi-dimensional mapping, real-time feeds, accessibility
- **3D Advanced**: Procedural textures, dynamic lighting, 360°/VR, math surfaces
- **Performance**: Profiling, frame rate optimization, Big-O analysis
- **Architecture**: Separation of concerns, modular design
- **AI**: Collaboration, prompt generation, attribution, AI assistance
- **Advanced Effects**: Shader effects, constraints, recursion, style transfer, narrative
- **Capstone**: Planning, implementation, documentation, iteration, portfolio
